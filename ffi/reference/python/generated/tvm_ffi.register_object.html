tvm_ffi.register_object#

tvm_ffi.register_object(type_key=None)[source]#

Register object type.

Parameters:

type_key (str | type | None, default: None) – The type key of the node

Return type:

Callable[[type], type] | type

Examples

The following code registers MyObject using type key “test.MyObject”

@tvm_ffi.register_object("test.MyObject")
class MyObject(Object):
    pass
