Namespace tvm::ffi::reflection::type_attr
=========================================


Type attribute names used by the reflection system. 




.. contents:: Contents
   :local:

Detailed Description
--------------------

Each constant names a :ref:`exhale_class_classtvm_1_1ffi_1_1reflection_1_1TypeAttrColumn` — a sparse, type-indexed slot that stores a ``Function`` or any other Any-compliant value.



- ``None`` to register a hook,

- ``TypeAttrColumn(type_attr::kFoo)[type_index]`` to look one up,
