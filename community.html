Join the TVM Community

Here are a few ways you can stay involved:

  • Mailing Lists

    As per Apache tradition, everything that happens in the community (also) happens in the mail-list.

    Mailing List
  • Discuss Forum

    We use discuss forum for general discussions and usage trouble shooting. We welcome all topics related to the TVM stack.

    Join The Discussion
  • Github

    We use our Github issue tracker for developer RFCs and roadmap discussion.

    Github
    • Mailing Lists

      As per Apache tradition, everything that happens in the community (also) happens in the mail-list. Here are the relevant mailing lists:

      • dev@tvm.apache.org development related activities.
      • All discuss forum thread and github issues with RFC COMMUNITY tags are automatically forwarded to dev@tvm.apache.org.
      • discuss-archive@tvm.apache.org archive of discuss forum.
    • Email dev-subscribe@tvm.apache.org to subscribe. Subscribe
