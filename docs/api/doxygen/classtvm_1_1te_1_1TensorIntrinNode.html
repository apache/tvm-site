Array<Buffer> tvm::te::TensorIntrinNode::buffers

Symbolic buffers of each output/input tensor buffers[0:len(inputs)] are buffers of the inputs. buffers[len(inputs):] are buffers of each output.

Note
When a field in Buffer is Var, it means we can be flexible wrt that field and Var can occur in body. When it is a constant, it means we can only take data in that shape.
