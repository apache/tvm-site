    Float(f64),

    NDArrayHandle(*mut c_void),
    Bytes(&'a TVMByteArray),
    Str(&'a CStr),

A borrowed TVMPODValue. Can be constructed using into() but the preferred way to obtain a ArgValue is automatically via call_packed!.
