Struct tvm::runtime::ObjectPtr

source · 
pub struct ObjectPtr<T>where
    T: IsObject,{
    pub ptr: NonNull<T>,
}
Expand description

A smart pointer for types which implement IsObject. This type directly corresponds to TVM’s C++ type ObjectPtr.

See object.h for more details.
