Detailed Description

Helper struct for return value of IdentifyMemCpy.

This helper struct is not strictly necessary, as IdentifyMemCpy could instead return a std::pair<BufferRegion, BufferRegion>. However, that would introduce ambiguity between the two unnamed regions.
