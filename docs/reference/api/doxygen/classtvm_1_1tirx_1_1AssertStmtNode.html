Detailed Description

Assert condition, if an error occurs, return the error message.

The error is described by:

  • error_kind: the error kind (e.g. "RuntimeError", "TypeError", "ValueError")
  • message_parts: an array of string fragments that are concatenated at runtime via TVMFFIErrorSetRaisedFromCStrParts. This enables string fragment reuse across multiple assertions to reduce binary size.
