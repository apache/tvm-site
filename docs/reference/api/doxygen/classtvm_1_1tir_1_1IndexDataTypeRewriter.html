Detailed Description

Data type rewriter for buffer indices.

Detect the components of buffer indices that should be considered for data type rewriting. This class doesn't perform actual rewriting of data types. During recursive visiting, the internal flags is_enabled_ and is_conditional_ are used to indicate whether the current expression is a buffer index or a conditional expression, which can be used in the sub-classes to implement different rewriting rules.
