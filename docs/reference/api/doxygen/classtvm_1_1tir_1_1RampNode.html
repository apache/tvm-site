Detailed Description

Construct a vector with lanes elements where its i-th element equals base + i * stride. This is useful to construct a index for a continuous vector load.

Examples:

  • ramp(0, 1, 3) = [0, 1, 2]
  • ramp(1, 2, 4) = [1, 3, 5, 7]
