constexpr const char* tvm::relay::attr::kPrimitive = "Primitive"

Mark the function as representing a sub-graph which is to be lowered or compiled as a unit. For example, the function may represent a kernel which TVM will lower to a PrimFunc. If present should be bound to Integer(1). May be accompanied by "Compiler", see below. The function body should be considered opaque by Relay, and many passes simply ignore these functions.
